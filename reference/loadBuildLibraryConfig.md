# buildLibrary

Load the build configuration from .buildLibrary. If the file does not
exist it is created.

## Usage

``` r
loadBuildLibraryConfig(lib = ".")
```

## Arguments

- lib:

  Path to the package

## Value

The configuration loaded from .buildLibrary as a list.
