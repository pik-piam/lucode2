# add EOF comment

remove EOF text from all files in the code

## Usage

``` r
removeEOF(path = ".", filetypes = c("inc", "prn", "gms"))
```

## Arguments

- path:

  path to the main folder of the model

- filetypes:

  file types the function should be applied to
